# 数值养成游戏 - 高三的逆袭

玩法：通过选择选项来指定角色行动来进行数值增加，或者是触发特殊事件。类似于《赛马娘》、《心跳回忆》。

游戏目前处于初期开发阶段，可以下载Ver. privarte-test (demo)版本进行试玩。
